# Purim Labs Architecture

## Core Areas

### Monitoring
PurimMonitor exposes lightweight server health and system statistics over HTTP.

### Security
SecureShare exposes a JWT-protected API for temporary file-share metadata.

### Networking
NetWatch provides command-line LAN diagnostics with explicit scan targets.

## Long-Term Direction
Purim Labs aims to build lightweight self-hosted infrastructure tooling focused on reliability, security, and operational visibility.
