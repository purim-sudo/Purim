# Security Policy

## Supported Versions
This repository is currently in active experimental development.

## Reporting Vulnerabilities
If you discover a security issue, please open a private report or create an issue describing the vulnerability.

Do not publish working exploit details in public issues.

## Goals
Purim Labs focuses on improving infrastructure reliability and secure software architecture.

## Operational Notes
- Set a unique `JWT_SECRET` of at least 32 characters before deploying SecureShare.
- Put production deployments behind HTTPS.
- Run NetWatch only on networks you own or have permission to assess.
